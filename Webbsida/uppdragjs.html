Tillbaka

Uppdrag 1

  1. Börja med att skapa ett nytt dokument som du kallar minsida.js och sparar på samma ställe som index.html och min.css
  2. Nu ska vi koppla ihop minsida.js och vår webbsida. Lägg till <script src="minsida.js" ></script> precis ovanför body-taggen längst ner på sidan.
  3. Nu behöver vi en knapp som vi kan leka med. Skapa en knapp under bilden på vår index.html sida som du döper till Test. <button onclick="">Test</button>
  4. Lägg till ett id="" på en av våra rubriker. T.ex: <h3 id="rubrik2">Jag heter Johan</h3>
  5. Nu lägger vi till lite javascript direkt i knappen. Vi lägger till koden i onclick="": onclick="document.GetElementById('rubrik2').innerHTML = 'Jag heter Kalle';"
  6. Glöm inte att spara innan ni testar.

Uppdrag 2

  1. Nu ska vi börja använda vår minsida.js för att skriva koden instället. Inne på minsida.js skriver vi:
    function nyText() {
    document.getElementById('rubrik2').innerHTML = 'Jag heter Kalle.';
    }
  2. Sedan går vi till vår index.html och byter ut vårt script i onclick till onclick="nyText()"
  3. Spara

Uppdrag 3

  1. Nu kan vi lägga till en knapp som kan byta bild också. Börja med att skapa en funktion som heter bytBild i minsida.js, smygkolla på hur du gjorde i förra uppgiften om du inte minns hur du gör.
  2. Gå till index.html och lägg till ett id="" på bildens img-tagg. Välj ett lämpligt id som beskriver vad bilden är till för t.ex. firstPagePic
    Skapa en ny knapp som heter Byt bild och som på onclick kör vår nya funktion.
  3. Nu kan vi gå till vår js-sida och lägga till lite kod. Inuti vår nya funktion lägger vi till följande kod:
    document.getElementById('firstPagePic').src="Adressen till nya bilden"
  4. Spara, ladda om testsidan och se om det fungerar.

Uppdrag 4

  1. Testa själv att göra en ny knapp och en ny funktion som byter tillbaka till den gamla bilden.
