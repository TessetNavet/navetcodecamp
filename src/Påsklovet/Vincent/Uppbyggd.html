Såhär ser en simpel länk/uppbyggnad ut i HTML!

  1. - !DOCTYPE html - detta berättar för webbläsaren att detta är en länk
  2. - html lang="en" - Detta förklarar vilket språk html ska vara i
  3. - Sedan används <head> för att för att säga vilken titel, scripts och annan information om dokumentet
  4. - Sen nästan sist av allt används <body> för att lägga in själva innehållet. Till exempel en bild.
  5. - Sist kan du skriva </html> som visar att du använder Html som språk.
